# YT-HACKER

Cyberpunk-style personal YouTube HD video & playlist downloader.

## Features
- HD video + audio merging (MP4)
- Playlist support
- Terminal-style cyberpunk UI
- Progress bar
- Keyboard sound FX
- Built with Python + yt-dlp
